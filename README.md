# Lunelle Sleep Pod — v0.1

AI-powered thermal regulation simulation for space sleep habitats.

## What it does
Simulates closed-loop thermal regulation in microgravity, modeling:
- Metabolic heat generation
- Sensor noise + moving average filtering
- State-based cooling activation
- Timeout fallback to prevent deadlock

## Built with
Python, NumPy, Matplotlib

v0.1 complete — CO2 module in progress
